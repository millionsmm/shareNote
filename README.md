## Introduction
 share 笔记遵循 * Material Design * 设计风格，支持 * MarkDown * 富文本编辑，可以帮你快速书写拥有自己独特的文字风格。支持追寻粉丝，添加笔记收藏，“不放过” 每一篇能触发你心灵的文章。我们注重于分享，无论是点滴灵感、待办清单、或是读书感想、项目资料，你都可以随时打开应用，随时记录，保存在本地亦或发布到网上与大家一起分享。
